# eteacher
UP teaching assistant

系统运行环境
JDK:1.7
tomcat:7.0
数据库：mysql 5.5

项目框架
后台：springMVC  + hibernate
前台：JQuery   , JQuery  Form Validator, 基于Bootstrap的table控件，freeMarker
